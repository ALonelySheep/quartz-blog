# Difference Between Function and Procedure


[Link](https://www.jianshu.com/p/2eaa094adf9e)


函数有1个返回值,而存储过程是通过参数返回的,可以有多个或者没有

#### E.g.

```cpp
ZZ x, a, n;  
x = InvMod(a, n);  _// functional form_  
InvMod(x, a, n);   _// procedural form_
```
